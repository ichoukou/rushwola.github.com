# jenkins 安装
## Tomcat方式安装

 - 下载jenkins的war包
 到jenkins官网下载war包即可，因为管网下载慢，也可以到别的地方下载，现在我是到csdn下载的jenkins2.22稳定版。
 
 - 运行
 将war包放到tomcat的　webapps下，运行tomcat即可。
 
 # 安装插件
 
    Maven插件 Maven Integration plugin
    发布插件 Deploy to container Plugin
    git插件 Git plugin
    svn插件 Subversion Plug-in
